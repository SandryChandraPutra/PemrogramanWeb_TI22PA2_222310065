Python

Python merupakan bahasa pemrograman komputer yang biasa dipakai untuk membangun situs, software/aplikasi, mengotomatiskan tugas dan melakukan analisis data.

JavaScript

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web yang interaktif. JavaScript dapat digunakan untuk menambahkan animasi, game dan fungsionalitas lainnya ke situs web.

Java

Java merupakan bahasa pemrograman berorientasi objek (Object Oriented Programming) yang relatif mudah dipelajari. Java juga dikenal memiliki moto “Write Once, Run Anywhere” artinya, Java mampu dijalankan di berbagai platform tanpa perlu disusun ulang menyesuaikan platformnya.

C#

Bahasa pemrograman C# atau dibaca C Sharp merupakan salah satu bahasa pemrograman berorientasi objek yang dikembangkan Microsoft. Bahasa C# adalah bahasa pemrograman sederhana yang digunakan untuk tujuan umum, digunakan untuk berbagai fungsi misalnya untuk pemrograman server-side pada website, membangun aplikasi desktop ataupun mobile, pemrograman game dan sebagainya.

C

Bahasa C adalah bahasa pemrograman tingkat menengah, karena lebih mudah dipelajari dan dimengerti namun tetap memiliki kemampuan yang tinggi. Bahasa pemrograman C sangat cocok untuk memperkenalkan konsep pemrograman untuk pemula. Bahasa C termasuk bahasa pemrograman general-purpose, artinya bahasa C bisa digunakan untuk membuat berbagai macam program komputer.

C++

Bahasa pemrograman C++ adalah bahasa pemrograman berorientasi objek yang populer karena menawarkan kinerja tinggi dan fleksibilitas. Bahasa C++ memiliki sintaks yang mirip dengan bahasa C, tetapi dengan beberapa fitur tambahan.

Go

Golang adalah singkatan dari Go Language (bahasa Go) yang merupakan bahasa pemrograman open-source yang dikembangkan oleh Google. Bahasa pemrograman Go dikenal ramah pemula karena memiliki sintaks yang sederhana dan mudah dipelajari.

R

Bahasa Pemrograman R adalah bahasa pemrograman yang dirancang untuk analisis statistika dan grafik, khususnya untuk orang yang bertugas menangani dan bekerja dengan data. Sebagai software yang banyak digunakan dalam pemrograman statistik, R memiliki banyak fitur dan fungsi yang membantu dalam menjalankan tugas sehari-hari, seperti manipulasi data dan data mining.

Swift

Swift adalah bahasa pemrograman yang kuat dan intuitif yang dikembangkan oleh Apple untuk iOS, macOS, dan platform Apple lainnya. Dasar bahasa ini adalah Objective-C dan dirancang untuk berjalan bersama framework Cocoa dari Apple. Melihat dari dasar bahasanya, bahasa C, C++ dan Objective-C dapat berjalan dengan Swift di program yang sama.

PHP

PHP (Hypertext Preprocessor) adalah bahasa pemrograman open-source yang umumnya digunakan untuk membangun aplikasi web dinamis dan interaktif. PHP dapat dijalankan pada server web dan dikombinasikan dengan HTML, CSS, dan JavaScript untuk membuat halaman web yang dinamis. PHP juga mendukung banyak jenis database, seperti MySQL, PostgreSQL, dan Oracle sehingga memungkinkan pengembang untuk membuat aplikasi web yang lebih kompleks dan fungsional.
